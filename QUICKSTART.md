- `/permits` - التصاريح

- `/map` - الخريطة التفاعلية
- `/reports` - البلاغات البيئية

## الأوامر المفيدة

```bash
# تثبيت التبعيات
pnpm install

# تشغيل التطوير
pnpm dev

# بناء للإنتاج
pnpm build

# تشغيل الإنتاج
pnpm start

# فحص الكود
pnpm lint
```
